# NaReT Lernplattform – geschützte Vorschau

Dieses Repository enthält ausschließlich **verschlüsselte** Build-Artefakte
(AES-256-GCM, Schlüssel per PBKDF2 aus einem Passwort abgeleitet).
Die Seite fragt unter https://naret-nachbarsprache-wird-reanimiert.github.io/ nach dem Passwort.

Quellcode und Deployment: privates Repo `lernplattform-with-mocks` (`npm run deploy`).
